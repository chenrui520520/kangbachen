KENBA 游戏官网 UI 文档
1️⃣ 设计风格
主题：暗黑美学
主色调：深紫色、黑色、暗红
字体：现代哥特风格字体 + 易读主文字字体
动画：
背景动态视频
鼠标悬停交互
奖励领取动画
页面滚动触发动画
组件风格：
卡片式展示任务、奖励、商城
发光按钮、高光边框
2️⃣ 页面组件设计
2.1 首页 / Hero 区
组件：
Logo / 游戏名称
登录按钮（邮箱 / X / 钱包）
背景动态视频
推荐任务/奖励/活动轮播卡片
交互：
登录按钮悬停发光
推荐卡片可滑动浏览
2.2 游戏背景 / 故事
组件：
视频播放器 / 背景视频
分段滚动文本 + 图片
交互动画（滚动触发）
交互：
鼠标悬停显示额外信息或动画
分段视频过渡特效
2.3 奖励系统 / 签到
组件：
签到按钮
当日奖励展示卡片
连续签到天数进度条
交互：
点击签到按钮触发奖励动画
奖励卡片闪光效果显示
2.4 任务系统
组件：
任务列表卡片（任务名称、类型、奖励）
任务进度条
完成任务按钮
交互：
点击任务完成按钮触发动画
动态更新任务状态
2.5 排行榜
组件：
积分排行榜表格
NFT 收集排行榜
交互：
可切换榜单类型
高亮当前用户行
2.6 限时活动
组件：
活动倒计时
活动任务列表
奖励展示
交互：
倒计时动态刷新
鼠标悬停显示奖励详细信息
2.7 积分兑换商城
组件：
商品卡片（NFT / 特殊奖励）
积分余额显示
兑换按钮
交互：
点击兑换触发动画
动态更新库存
2.8 社区 / 社交
组件：
X / Discord / Telegram 按钮
社区活动推荐轮播
2.9 页脚
组件：
隐私政策
免责声明
风格：
黑色背景 + 白色/灰色文字
4️⃣ 推荐前端框架和组件（供 Cursor 使用）
框架：
React + Next.js（SPA + SSR + SEO）
UI 组件库：
Framer Motion（动画效果）
React Player（视频播放）
React Slick / Swiper（推荐轮播）
状态管理：
Redux Toolkit 或 Zustand
区块链钱包集成：
Web3Modal + Ethers.js（MetaMask、OKX、Coinbase 等）
